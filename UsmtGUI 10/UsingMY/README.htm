Using the My Namespace

Available only in VB.NET, the My namespace collects commonly accessed objects in one convenient location. The My namespace includes:
  • My.Application: Application information, including assembly information
  • My.Computer: Access to various computer resources, including drives, network identity, and more
  • My.Forms: A list of the forms defined in the application
  • My.Resources: Application resources, often used for localization
  • My.Settings: Application settings, e.g., database connection information
  • My.User: Information about the current user
  • My.WebServices: A list of the web services used in the application
This sample contains one form that demonstrates several features of the My namespace. Two empty forms are included for demonstrating My.Forms.

This sample requires the following to be installed:
  • Visual Studio 2005
